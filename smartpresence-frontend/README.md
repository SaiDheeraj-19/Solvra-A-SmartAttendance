# SmartPresence - Luxury Frontend

Next.js 15 + TypeScript + TailwindCSS frontend for SmartPresence — a luxury, Modern Huntsman-inspired design system for attendance (Geo-fence, QR, Face ID).

Tech stack

- Next.js 15 (App Router)
- TypeScript
- TailwindCSS
- Framer Motion

Getting started

1. cd into project:

```bash
cd smartpresence-frontend
```

2. Install dependencies:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```

Design system

- Colors: #f9f8f7 (background), #1b1b1b (text), #bfa47a (bronze accent)
- Typography: Playfair Display (serif), Inter (sans-serif)
- Motion: Framer Motion used for subtle interactions

Project layout

- /src/app — App Router pages and layouts
- /src/components — Reusable UI components (ui primitives)
- /src/lib — Utilities
- /public — Static assets

Environment

- Copy `.env.example` to `.env.local` and update values as needed.
